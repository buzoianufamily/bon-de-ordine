Fonturi gazduite local (self-hosted)
====================================

Motiv: conformitate GDPR (niciun IP de vizitator nu pleaca la Google Fonts)
si independenta de retea (aplicatia functioneaza si offline / pe retele inchise).

Familii incluse (subseturi latin + latin-ext, format woff2 variabil):
  - Manrope               — Copyright The Manrope Project Authors
  - Bricolage Grotesque   — Copyright The Bricolage Grotesque Project Authors

Textul complet al licentei: https://openfontlicense.org
Sursa fonturilor: https://fonts.google.com (Manrope, Bricolage Grotesque).

OFL permite utilizarea, includerea si redistribuirea fonturilor in produse
software, inclusiv comerciale. Fonturile nu se vand separat.
